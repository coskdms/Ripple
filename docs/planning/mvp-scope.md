# MVP 범위

## 만든다 (Must have)
1. **결정 기록** — 4항목 (상황/결정/이유/감정)
2. **결정 목록 보기** — 시간순
3. **결정 상세 보기** — 펼쳐서 다시 읽기
4. **회고** — 프리셋 4종(일간/주간/월간/연간) 켜고 끄기
   - "N개월 전 오늘, 당신은 이런 결정을 했어요" (감성 핵심)
5. **인증** — 구글 소셜 로그인

## 나중에 (v2+)
- 회고 커스텀 주기 (3일마다 등)
- 회고 알림(리마인드)
- 결정 수정 (원본 보존 방식)
- 목표/가치관 설정
- 결정 숙성도
- AI 패턴 분석 / RAG
- 포기한 선택지 추적
- 익명 공유
- 결과 평가
